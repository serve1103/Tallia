## 계산 과정 설정 공통

### 사용자 정의 단계
사용자가 사용자 정의 계산 단계을 추가할 수 있다.
비전공자를 위해 **템플릿 방식**을 기본으로, 고급 사용자용 수식 입력도 지원.

**사용자 정의 단계 템플릿:**

| 템플릿 | 설명 | 파라미터 예시 |
|--------|------|---------------|
| 가산점 부여 | 조건 충족 시 점수 가산 | 조건: 점수 >= 80, 가산: +5점 |
| 비율 조정 | 점수에 비율 곱하기 | 비율: 0.6 |
| 구간 변환 | 점수 구간별 다른 값 매핑 | 90~100→A, 80~89→B, ... |
| 상한/하한 제한 | 점수 범위 제한 | 최소: 0, 최대: 100 |
| 직접 수식 (고급) | 변수 기반 수식 직접 입력 | score * 0.6 + bonus |

- 각 템플릿은 한글 가이드 + 예시가 포함된 입력 폼 제공
- 직접 수식은 사용 가능한 변수 목록(score, maxScore, count 등)을 드롭다운으로 제공
- 저장 전 테스트 값으로 결과 검증 필수
- **보안**: 직접 수식은 허용 변수/함수 화이트리스트만 사용 가능, 서버에서 샌드박스 환경으로 실행 (코드 인젝션 방지)

### 후처리 단계 (모든 유형 공통)

| 단계 | 입력 | 출력 | 설명 |
|------|------|------|------|
| 항목별 과락 판정 | [항목×위원] 또는 [항목별 점수] | 동일 + 과락 플래그 | 항목별 과락 기준 체크 (위원 총점 방식 진입 전 또는 원본 데이터 기반으로 판정) |
| 전체 과락 판정 | 단일 점수 | 동일 + 과락 플래그 | 전체 점수 기준 과락 체크 |
| 만점 기준 환산 | 단일 점수 | 단일 점수 (원점수) | 100점 만점 기준으로 환산 → 원점수 산출 |
| 환산 만점 적용 | 단일 점수 (원점수) | 단일 점수 (환산점수) | 환산 만점 입력 (예: 500) → 원점수 × (환산만점 / 100) = 환산점수 (예: 85 × 5 = 425점) |

## A. 위원 평가

- **대상**: 서류, 면접, 실기
- **입력**: 평가위원의 항목별 점수 또는 등급 (엑셀 업로드)
- **특징**:
  - 면접일자/조별로 위원 수가 다를 수 있음 (최대 위원 수 기준 열 생성, 적은 날은 빈칸)
  - 항목 구조: 대항목 → 소항목(optional) 계층 지원
  - 계산 과정을 단계별로 자유롭게 구성

### 설정 항목

#### 기본 정보
- 평가명
- 최대 평가위원 수 (엑셀 열 수 결정, 실제 위원 수는 행마다 다를 수 있음)

#### 위원 데이터 유형 (엑셀에 들어오는 값의 형태)
- **점수**: 위원이 항목별 점수를 매긴 데이터
- **등급**: 위원이 항목별 등급(A/B/C 등)을 매긴 데이터
  - 항목별 등급→점수 점수 변환표 설정 (항목마다 다를 수 있음)
  - 예: 논리력 A=50, B=40, C=30 / 인성 A=100, B=80, C=60

#### 항목 구조
- **대항목** 추가/삭제 (예: 인성, 전공적합성)
- 대항목별 **소항목** 추가/삭제 — optional (예: 인성 → 성실성, 협동심)
- 소항목 없이 대항목만 사용 가능
- 항목별 설정: 만점, 가중치, 과락 기준

### 계산 과정 설정

계산 과정을 단계별 계산 단계 조합으로 구성한다.
관리자가 [단계 추가] 버튼으로 연산을 순서대로 쌓는 방식.

**데이터 형태 변화:**
초기 데이터는 [항목 × 위원] 2차원 행렬. 계산 과정은 이 행렬을 단계적으로 축소하여 최종 단일 점수를 산출한다.

```
초기: [항목 × 위원] 행렬
         논리력  표현력  인성
위원1:    40     30     50
위원2:    45     35     45
위원3:    35     25     55
```

**위원 총점 방식 (위원 차원 먼저 축소)**
위원별로 항목을 합산한 뒤, 위원 총점을 비교/집계
```
[위원별 항목 합산] → [120, 125, 115]  (위원별 총점)
→ [최고 위원 제외] → [120, 115]       (125점 위원 제거)
→ [위원 평균] → 117.5                 (단일 점수)
```

**항목별 계산 방식 (항목 차원 유지하며 위원 축소)**
각 항목별로 위원 점수를 집계한 뒤, 항목을 합산
```
[항목별 최고점 제외] → 논리력[40,35], 표현력[30,25], 인성[50,45]
→ [항목별 위원 평균] → 논리력37.5, 표현력27.5, 인성47.5
→ [항목 합산] → 112.5                 (단일 점수)
```

**사용 가능한 계산 단계:**

*전처리 단계 (등급 입력 시에만 표시)*

| 단계 | 입력 | 출력 | 설명 |
|------|------|------|------|
| 등급→점수 변환 | [항목×위원] 등급 | [항목×위원] 점수 | 항목별 점수 변환표로 점수 변환 |

*위원 총점 방식 단계*

| 단계 | 입력 | 출력 | 설명 |
|------|------|------|------|
| 위원별 항목 합산 | [항목×위원] | [위원 총점 배열] | 각 위원의 모든 항목 점수를 합산 |
| 위원별 항목 가중합산 | [항목×위원] | [위원 총점 배열] | 각 위원의 항목 점수에 가중치 적용 후 합산 |
| 가상 위원 추가 | [위원 총점 배열] | [위원 총점 배열 +1] | 가상 점수 생성 추가 (최고최저 평균 / 전체 평균 / 지정값) |
| 최고 위원 제외 | [위원 총점 배열] | [위원 총점 배열 -1] | 총점이 가장 높은 위원 제거 |
| 최저 위원 제외 | [위원 총점 배열] | [위원 총점 배열 -1] | 총점이 가장 낮은 위원 제거 |
| 위원 평균 | [위원 총점 배열] | 단일 점수 | 남은 위원 총점의 평균 |
| 위원 합산 | [위원 총점 배열] | 단일 점수 | 남은 위원 총점의 합산 |

*항목별 계산 방식 단계*

| 단계 | 입력 | 출력 | 설명 |
|------|------|------|------|
| 항목별 가상 위원 추가 | [항목×위원] | [항목×위원 +1] | 각 항목에 대해 가상 위원 점수 추가 |
| 항목별 최고점 제외 | [항목×위원] | [항목×위원 -1] | 각 항목에서 최고 위원 점수 제거 |
| 항목별 최저점 제외 | [항목×위원] | [항목×위원 -1] | 각 항목에서 최저 위원 점수 제거 |
| 항목별 위원 평균 | [항목×위원] | [항목별 점수] | 각 항목의 위원 점수를 평균 |
| 항목별 위원 합산 | [항목×위원] | [항목별 점수] | 각 항목의 위원 점수를 합산 |
| 가중치 적용 | [항목별 점수] | [항목별 점수] | 항목별 가중치 반영 |
| 소항목 합산→대항목 | [소항목별 점수] | [대항목별 점수] | 소항목 합산으로 대항목 산출 |
| 소항목 가중합산→대항목 | [소항목별 점수] | [대항목별 점수] | 소항목 가중합산으로 대항목 산출 |
| 항목 합산 | [항목별 점수] | 단일 점수 | 항목 점수 합산 |
| 항목 평균 | [항목별 점수] | 단일 점수 | 항목 점수 평균 |

*후처리 단계은 "계산 과정 설정 공통" 섹션 참조*

**위원 수별 조건부 계산 과정:**
면접일자별로 위원 수가 다를 수 있으므로, 위원 수별로 다른 계산 과정을 설정 가능.
시스템이 각 행의 실제 위원 수(빈칸이 아닌 셀 수)를 자동 감지하여 해당 계산 과정 적용.

```
위원 5명일 때 (위원 총점 방식):
  [위원별 항목 합산] → [최고 위원 제외] → [최저 위원 제외] → [위원 평균] → [전체 과락 판정] → [만점 기준 환산] → [환산 만점 적용]

위원 4명일 때 (위원 총점 방식):
  [위원별 항목 합산] → [가상 위원 추가(최고최저 평균)] → [최고 위원 제외] → [최저 위원 제외] → [위원 평균] → [전체 과락 판정] → [만점 기준 환산] → [환산 만점 적용]

위원 3명일 때 (항목별 계산 방식):
  [항목별 위원 평균] → [가중치 적용] → [항목 합산] → [전체 과락 판정] → [만점 기준 환산] → [환산 만점 적용]
```

4명 구체 예시 (위원 총점 방식):
```
         논리력  표현력
위원1:    40     30     → 합계 70
위원2:    45     35     → 합계 80
위원3:    35     25     → 합계 60
위원4:    50     20     → 합계 70

[위원별 항목 합산] → [70, 80, 60, 70]
[가상 위원 추가(최고최저 평균)] → (80+60)/2 = 70 → [70, 80, 60, 70, 70]
[최고 위원 제외] → 80 제거 → [70, 60, 70, 70]
[최저 위원 제외] → 60 제거 → [70, 70, 70]
[위원 평균] → 70
[만점 기준 환산] → 70 / 100 × 100 = 70 (원점수)
[환산 만점 적용] → 70 × 5 = 350 (환산점수, 500점 만점)
```

**단계 유효성 검증:**
각 단계은 입력/출력 데이터 형태가 정의되어 있으므로, 이전 단계의 출력과 다음 단계의 입력이 맞지 않으면 경고/차단.
- 위원 총점 방식 단계과 항목별 계산 방식 단계은 혼용 불가 (입력 데이터 형태가 다름)
- [만점 기준 환산] 단계이 계산 과정에 없으면 경고
- 등급 데이터인데 [등급→점수 변환] 없이 진행하면 경고
- 최종 출력이 단일 점수가 아니면 경고

### 엑셀 양식

**점수 입력** (예: 최대 위원 5명, 대항목 2개, 소항목 있음)
```
수험번호 | 성명 | 인성_성실성_위원1 | 인성_성실성_위원2 | ... | 인성_성실성_위원5 | 인성_협동심_위원1 | ... | 전공_소항목1_위원1 | ...
```
- 위원 수가 적은 행은 뒷 열을 빈칸으로 남김 → 시스템이 빈칸 수로 실제 위원 수 자동 감지 → 해당 위원 수 계산 과정 적용
- 면접조 열 불필요 (위원 수 기반 자동 분기)
- 등급 입력 시 셀 값은 A, B, C 등

## B. 자동 채점 (필답고사)

- **대상**: 필답고사
- **입력**: 정답지 + 수험생 마킹 답안
- **처리**: 정답 대조 → 문항별 배점 적용 → 총점 → 환산

### 설정 항목
- 기본 정보: 평가명, 과목 수, 만점
- 과목별 설정:
  - 과목명, 문항 수, 문항별 배점, 과목 만점
  - 과목별 과락 기준
  - **시험유형** 설정 (예: A형, B형) — 유형별로 정답지와 배점이 다를 수 있음
    - 유형 간 문항 수가 다를 경우: 유형별 별도 시트로 업로드
  - 유형별 정답지 등록 (문항별 정답, 복수정답 여부)
  - **출제 오류 처리**: 특정 문항에 대해 사후 처리 가능
    - 전원 정답 처리: 해당 문항을 모든 수험생 정답으로 간주
    - 배점 제외: 해당 문항을 총점 계산에서 제외하고 나머지 기준으로 재산출
- 전체 과락 기준
- 채점 결과 검증용 감사 로그

### 계산 과정 설정
채점 → 과목 합산 → 과락 판정 → 만점 기준 환산 → 환산 만점 적용 등 단계 조합
- ※ 단계별 입력/출력 상세 정의는 기술 설계 단계에서 A유형과 동일한 형식으로 정의

### 엑셀 양식

**유형 동일 문항 수** (예: 과목 1개, 문항 50개)
```
수험번호 | 성명 | 시험유형 | 1번 | 2번 | 3번 | ... | 50번
```

**유형별 문항 수 다름** (유형별 별도 시트)
```
[A형 시트] 수험번호 | 성명 | 1번 | 2번 | ... | 50번
[B형 시트] 수험번호 | 성명 | 1번 | 2번 | ... | 45번
```

## C. 문항별 채점 (논술)

- **대상**: 논술
- **입력**: 문항별 점수 (사람이 채점한 결과)
- **처리**: 계산 기준 적용 → 환산

### 설정 항목
- 기본 정보: 평가명, 만점
- **채점위원 수** (optional): 복수 채점위원이 같은 문항을 채점하는 경우
  - 위원별 점수를 받아 집계 (평균 / 최고최저 제외 등 — A유형과 동일한 위원 집계 단계 재활용)
  - 채점위원 없이 단일 점수만 받는 것도 가능
- 문항 구조: **대문항 → 소문항** 계층 지원
  - 대문항 추가/삭제 (예: 1번, 2번, 3번)
  - 대문항별 소문항 추가/삭제 (예: 1-1, 1-2, 1-3)
  - 소문항 없이 대문항만 사용도 가능
- 문항별 설정: 만점, 배점, 가중치
- 과락 설정:
  - 소문항별 과락 기준 (optional)
  - 대문항별 과락 기준 (optional)
  - 전체 과락 기준
- 대문항 점수 산출: 소문항 합산 / 가중합산
- 전체 계산 기준 (합산 / 가중합산 — 특수 계산은 사용자 정의 단계으로 대체)

### 계산 과정 설정
위원 집계(해당 시) → 소문항 합산 → 대문항 합산 → 과락 판정 → 만점 기준 환산 → 환산 만점 적용 등 단계 조합
- ※ 단계별 입력/출력 상세 정의는 기술 설계 단계에서 A유형과 동일한 형식으로 정의

### 엑셀 양식

**단일 채점** (예: 대문항 2개, 소문항 포함)
```
수험번호 | 성명 | 1-1 | 1-2 | 1-3 | 2-1 | 2-2
```

**복수 채점위원** (예: 위원 2명, 다중행)
```
수험번호 | 성명 | 위원 | 1-1 | 1-2 | 1-3 | 2-1 | 2-2
001     | 홍길동 | 위원1 | 13  | 14  | 8   | 25  | 28
001     | 홍길동 | 위원2 | 12  | 15  | 9   | 27  | 26
002     | 김철수 | 위원1 | 10  | 12  | 7   | 20  | 22
002     | 김철수 | 위원2 | 11  | 13  | 8   | 22  | 24
```

## D. 점수 변환표

- **대상**: 자격증, 공인어학, 전적대학성적, 실적, (비교과 — 출결/봉사 등 필요 시)
- **입력**: 유형에 따라 다름
  - 자격증: 자격유형 + 급수
  - 공인어학: 어학유형 + 점수 또는 등급
  - 전적대학: 백분위 또는 평균평점
  - 실적: 실적유형 + 등급/수준
- **처리**: 대학별로 설정한 점수 변환표에서 점수 조회

### 설정 항목
- 기본 정보: 평가명, 매핑 유형 선택 (자격증/어학/전적대학/실적/기타)
- 점수 변환표 편집:
  - 자격증: (유형, 급수) → 점수 테이블
  - 공인어학: (어학유형, 점수구간) → 점수 테이블
  - 전적대학: (평점/백분위 구간) → 점수 테이블
  - 실적: (유형, 등급) → 점수 테이블
  - 기타: 범용 (조건1, 조건2) → 점수 테이블
- 점수 변환표 입력 방식:
  - **엑셀 업로드**: 점수 변환표을 엑셀로 업로드 (행이 많을 때 편리)
  - **화면 직접 편집**: 업로드 후 화면에서 확인/수정 가능
  - 두 방식 병행 — 업로드로 초기 데이터 구성, 화면에서 미세 조정
- 만점 설정
- 전체 과락 기준
- **매칭 실패 처리**: 점수 변환표에 없는 값이 업로드되면 해당 행을 오류로 표시, 업로드 시 실패 인자 알람
- **복수 보유 시**: 사용자가 인정한 건만 업로드 (엑셀에 1건만 기재, 복수 건 선별은 업로드 전 대학 측에서 처리)

### 계산 과정 설정
매핑 조회 → 과락 판정 → 만점 기준 환산 → 환산 만점 적용 등 단계 조합
- ※ 단계별 입력/출력 상세 정의는 기술 설계 단계에서 A유형과 동일한 형식으로 정의

### 엑셀 양식

**자격증**
```
수험번호 | 성명 | 자격유형 | 급수
```

**공인어학**
```
수험번호 | 성명 | 어학유형 | 점수
```

## 공통 기능

- **평가 복사**: 기존 평가의 모든 설정을 복사하여 새 평가 생성 (매년 유사한 평가 재사용)
- **샘플 미리보기**: 모든 유형에서 설정 완료 후 샘플 데이터로 각 단계별 중간 결과 + 최종 결과 검증

## 흐름 예시

### 예시 1: 면접 — 위원 총점 기준, 위원 수 가변 (A. 위원 총점 방식)
```
1. 평가 생성 → 유형: 위원 평가, 학년도: 2026, 전형: 수시
2. 데이터 유형: 점수
3. 최대 위원 수: 5명
4. 항목: 인성(만점100), 전공적합성(만점100)
5. 계산 과정 (위원 수별, 위원 총점 방식):
     5명: [위원별 항목 합산] → [최고 위원 제외] → [최저 위원 제외] → [위원 평균] → [전체 과락 판정] → [만점 기준 환산] → [환산 만점 적용]
     4명: [위원별 항목 합산] → [가상 위원 추가(최고최저 평균)] → [최고 위원 제외] → [최저 위원 제외] → [위원 평균] → [전체 과락 판정] → [만점 기준 환산] → [환산 만점 적용]
     3명: [위원별 항목 합산] → [위원 평균] → [전체 과락 판정] → [만점 기준 환산] → [환산 만점 적용]
6. 엑셀 양식 다운로드 (10개 점수 열) → 업로드 → 계산 → 결과 조회
```

### 예시 2: 면접 — 항목별 기준, 등급 입력 + 소항목 (A. 항목별 계산 방식)
```
1. 평가 생성 → 유형: 위원 평가
2. 데이터 유형: 등급
3. 최대 위원 수: 3명
4. 대항목/소항목:
     인성 → 성실성(A=30,B=20,C=10), 협동심(A=30,B=20,C=10)
     전공 → 이해도(A=40,B=25,C=10)
5. 계산 과정 (항목별 계산 방식):
     [등급→점수 변환] → [항목별 위원 평균] → [소항목 합산→대항목] → [가중치 적용] → [항목 합산] → [전체 과락 판정] → [만점 기준 환산] → [환산 만점 적용]
6. 처리 예시 (지원자 1명):
     [등급→점수 변환]
       성실성: 위원1=A(30), 위원2=B(20), 위원3=A(30)
       협동심: 위원1=B(20), 위원2=A(30), 위원3=B(20)
       이해도: 위원1=A(40), 위원2=B(25), 위원3=A(40)
     [항목별 위원 평균]
       성실성: 26.67 / 협동심: 23.33 / 이해도: 35.00
     [소항목 합산→대항목]
       인성: 26.67 + 23.33 = 50.00 / 전공: 35.00
     [항목 합산]
       합산: 85.00
     [만점 기준 환산]
       85.00 / 100 × 100 = 85.00 (원점수)
     [환산 만점 적용]
       85.00 × 5 = 425.00 (환산점수, 500점 만점)
```

### 예시 3: 필답고사 (B. 자동 채점)

2. 과목: 영어 (50문항, 문항별 배점 설정)

## 검증
- A유형: 위원 수별 조건부 계산 과정 동작 검증, 등급 입력 + 소항목 계층 검증
- B유형: 복수정답, 출제오류(전원정답/배점제외), 시험유형별 채점 검증
- C유형: 대문항/소문항 계층 계산, 복수 채점위원 집계 검증
- D유형: 각 매핑 유형별(자격증/어학/전적대학) 변환, 매칭 실패 처리 검증
- 공통: 소수점 처리, 재업로드 롤백, 평가 복사, 원점수+환산점수 산출, 사용자 정의 단계 검증
